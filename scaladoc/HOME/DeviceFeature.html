DeviceFeature 

case class DeviceFeature[A <: Component { ... /* 2 definitions in type refinement */ }](deviceName: String, featureTitle: String, initialValue: String, setterComponent: A, updateType: String) extends Label with EditableFeature with Product with Serializable

A device feature that can be changed either by user or profile.

A

Component with the ability to return the new value set by user and to set it (if needed) by Coordinator via EditableFeature

deviceName

name of the device this feature belongs to.

featureTitle

name of such feature

initialValue

current value to display

setterComponent

component through which users can change feature val

updateType

type of update(Msg) this feature sends to coordinator once a click is performed on this label, setterComponent is opened and user can change feature value; if so, such change is sent to coordinator via Promise. When the promise completes, this feature value is updated.
